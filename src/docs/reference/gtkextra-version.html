Description

This section describes the variables and functions available to test the version of the GTK-EXTRA library in use.

 


gtkextra_major_version

extern const guint gtkextra_major_version;

The major version number of the GTK-EXTRA library. (e.g. in GTK-EXTRA version 0.99.17 this is 0.)

gtkextra_minor_version

extern const guint gtkextra_minor_version;

The minor version number of the GTK-EXTRA library. (e.g. in GTK-EXTRA version 0.99.17 this is 99.)

gtkextra_micro_version

extern const guint gtkextra_minor_version;

The micro version number of the GTK-EXTRA library. (e.g. in GTK-EXTRA version 0.99.17 this is 17.)

gtk_extra_check_version ()

gchar *  gtk_extra_check_version  (guint required_major,
                                   guint required_minor,
                                   guint required_micro);

Checks that the GTK-EXTRA library in use is compatable with the given version.

required_major : the required major version.
required_minor : the required major version.
required_micro : the required major version.
Returns : NULL if the GTK-EXTRA library is compatable with the given version, or a string describing the version mismatch.
